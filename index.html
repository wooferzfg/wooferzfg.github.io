Projects

I've worked on some cool projects by myself and with friends. Here are some of the things I've made in my free time.

LiveSplit
LiveSplit is a highly-customizable timer used for speedrunning video games, and it is an open source .NET application. LiveSplit currently has over 400,000 downloads.
Traveler
This is a game based on the Traveling Salesman Problem in which the player must constantly maintain an efficient round trip path between all of the nodes on the screen.
The Wind Waker Randomizer Tracker
This web application allows the user to track their progress while playing The Wind Waker Randomizer. They can set which items they've already obtained and which locations they've already checked.
