This puzzle was created for zetamath's challenge to set a Sudoku with no 5s.

Rules

Place the digits 1-4 and 6-9 exactly once in every row, column, and region.

Each purple line contains a set of consecutive and non-repeating digits, in any order. (The digits 4 and 6 are NOT consecutive.)
